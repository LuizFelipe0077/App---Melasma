Arquitetura do Sistema

Construção modular e em camadas inspirada no DDD e Clean Architecture.

Arquitetura Hexagonal

O backend foi projetado desacoplando totalmente a regra de negócio do motor de persistência. A migração para PostgreSQL ou Firestore pode ser realizada apenas alterando a classe adaptadora do repositório, sem alterar um único Use Case.

Configurações do Sistema

SystemConfiguration.js 
export const SystemConfiguration = {
  CHECKIN_WINDOW_TOLERANCE_MINUTES: 60, // Tolerância de ±60 minutos
  MAX_DAYS_TREATMENT: 90,              // Janela de protocolo
  MAX_LOGIN_ATTEMPTS: 5,                // Bloqueio temporário
  LOGIN_LOCKOUT_MINUTES: 15             // Lockout de 15 minutos
};
